About

While cooking is fun, I also realized that I enjoy coding. Making things is so fun! I once wrote a script to organize files for a software application. Portfolio

Recipe Sharing App

Stack: React, Node.js, Express, PostgreSQL, Mocha/Chai, Jest/Enzyme, JWT, and CSS

An application to share recipes for those who like to see others' recipes and share their own. Made to give users a place to discover or talk about recipes. Please take a look at the GitHub client and server repositories or the live site.

Logger App

Stack: React, Node.js, Express, PostgreSQL, Mocha/Chai, Jest/Enzyme, JWT, and CSS

An application for logging start and stop times. Useful for hourly workers or people who log times. Made to organize and view logs. Logs are shown in the user's time zone (but stored in UTC). Please take a look at the GitHub client and server repositories or the live site.

Stack: React, Node.js, Express, PostgreSQL, Mocha/Chai, JWT, and CSS

An application for people who want to learn french words using spaced repetition as the quizzing method. Made to implement a linked list and sorting algorithm. client and server repositories or the live site.
